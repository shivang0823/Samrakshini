# **App Name**: Samrakshni

## Core Features:

- Panic Button Activation: Instantly send the user's live location to emergency contacts and local authorities.
- Community Incident Reporting: Allow users to report incidents, which populates the map to indicate 'red zones'.
- Safety Tips Sharing: Users can share and read safety tips relevant to their area or general safety.
- Interactive Safety Map: Display a map with marked unsafe zones based on incident reports and highlight safe locations like police stations.
- Reputation System: Implement a system to verify trusted contributors who actively share reliable information and tips.
- Emergency Contact Management: Allow users to add, remove, and manage emergency contacts within the app.

## Style Guidelines:

- Primary color: Deep calming blue (#3F51B5), provides a sense of security and trust.
- Background color: Very light blue (#E8EAF6), provides a calm, unobtrusive backdrop.
- Accent color: Warm orange (#FF9800) for alerts, highlights, and important CTAs to immediately draw user attention.
- Body and headline font: 'PT Sans' (sans-serif) for a clean, accessible, and modern look.
- Use clear, intuitive icons from a consistent set (e.g., Material Design Icons) for actions and navigation.
- A clean and intuitive layout, making key features easily accessible; the panic button should be prominent.
- Subtle transitions and animations to provide feedback on user interactions, enhancing the app's usability and perceived safety.
